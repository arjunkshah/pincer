Installation

Download the zip, load it as an unpacked extension, then add your Groq API key.

Download

Grab the latest build and keep it on your machine.

Download .zip
Install steps
  1. Unzip the file.
  2. Open chrome://extensions.
  3. Enable Developer mode (top right).
  4. Click "Load unpacked" and select the unzipped folder.
  5. The Pincer Setup tab opens - paste your Groq API key and accept the terms.
Terms & privacy

The extension stores your API key locally and sends selected page text to Groq for AI rewrites.

Privacy Policy
  1. Your settings and API key are stored locally in your browser.
  2. We do not sell or share personal data.
  3. Selected page text is sent to Groq to generate AI responses.
  4. Your API key is used only to authenticate those requests.
  5. We do not retain or log processed text.
  6. Remove data by clearing extension storage or uninstalling.
